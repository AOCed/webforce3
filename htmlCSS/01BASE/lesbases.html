Les balises DIV :

La balise DIV est une balise neutre de type block. Elle me servira pour structurer ma page.

Les balise blocs agissent sur un axe vertical et donc génèrent un retour à la ligne.
Je pourrais toute fois changer leur comportements avec du CSS.

Les balises Inline agissent sur un axe horizontal et donc ne génèrent aucun retour à la ligne.
Je pourrais toutefois changer leurs comportements avec du CSS.

Les Formulaires :

  • La balise FORM permet de créer un formulaire.
  • name: permet de donner un nom au formulaire.
  • action: Cet attribut est ogligatoire ! il permet de renseigner le script, la page vers laquelle l'utilsateur sera redirigé lors de l'envoi du formualire. Cette page va recevoir l'ensemble des données saisies.
  • method: permet de préciser quelle méthode [GET | POST] doit etre utilisé par le navigateur pour transmettre les données saisies vers la page "action".
